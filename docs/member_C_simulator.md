# 成员 C：温度仿真与实时曲线模块

## 负责模块

五通道温度仿真、稳定判定、实时曲线和温度漂移计算。

## 主要文件

| 文件 | 说明 |
|---|---|
| `services/simulator.py` | 温度仿真算法 |
| `models/sensor_data.py` | 温度数据模型 |
| `ui/main_window.py` | Matplotlib 曲线显示 |

## 需要实现的功能

- 炉温1、炉温2逐渐升到 750℃。
- 表面温和中心温在记录阶段缓慢上升。
- 校准温接近炉温1。
- 每 800ms 更新温度。
- 温度稳定后进入 Ready。
- 实时曲线显示 TF1、TF2、TS、TC。

## 与其他成员的接口

| 成员 | 接口 |
|---|---|
| B | `TestController.tick()` 调用 `SensorSimulator.update()` |
| D | 报告模块读取 `record_samples` |
| E | 主界面读取最新温度刷新显示 |

## 测试方法

点击开始升温，观察炉温是否上升至 750℃ 附近；开始记录后观察表面温和中心温是否缓慢上升。

## 最终交付物

仿真曲线平滑，状态稳定判定可靠，曲线能实时刷新。
